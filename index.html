Práctica 3:Autentificación con Google

La práctica consiste en utilizar el ejemplo de uso de OmniAuth en https://github.com/crguezl/omniauth-google-oauth2-sample y partimos de ese ejemplo configurando el ID Client y Secret Key Después al crear una nueva aplicación en Google's Console nos genera un ID de cliente y su secreto de cliente. Posteriormente debemos habilitar las APIs de Google para que se puede autentificar y mostrar el resultado.

LocalHost

La primera prueba que debemos hacer es en local para ver que la aplicación funciona. Para ello debemos crear un nuevo ID de cliente,especificando la ruta con la que queremos que se muestre nuestra aplicación.En nuestro caso vamos a especificar http://localhost:9292.

Despligue en Heroku

Lo primero que debemos hacer es crear una aplicación en heroku,para posteriormente generar una nueva id de cliente y copiar la ruta de la misma en el Google's Console.
